Screen 5 — Comparison modal notes
·Per-row decision: Each difference has its own Approve / Reject toggle. The illustrated state shows 5 approved and 3 rejected — operators are not forced to apply all or nothing.
·Live counter: The summary bar updates as decisions change, showing the current approved and rejected counts at a glance.
·Recalculation only on confirm: Clicking "Apply approved & recalculate" applies only approved differences as adjustments, then recalculates the run before presenting it for approval.
·Dismiss path: Closing or clicking "Dismiss" rejects all differences. The run proceeds to approval without any adjustments or recalculation.
·Outcome recorded: The comparison result, each decision (approved/rejected), and any applied adjustments are recorded against this run regardless of path taken.
